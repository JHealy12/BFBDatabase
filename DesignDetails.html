Genre and Expectation

Black flag battles is a multiplayer competitive game that allows players to compete against each other in a top down semi-realistic setting. The game will allow for players to be pinned against each other in an accurate battle with a range of different ships. The game should somewhat simulate pirate battles from a different perspective.

Level Design

Black Flag Battles has two game modes an offline which will allow for players to select different ship sizes to fight one another. Online which will allows players from around the world to fight using the same boat.

Environment

Every map will have a blue ocean to sail on. There will be a combination of lethal and non lethal objects in the maps. Lethal Objects Sea mines, Floating gun powder barrels and rocks. The non lethal objects are the islands.
